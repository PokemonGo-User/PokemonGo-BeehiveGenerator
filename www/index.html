How To

Create beehive

  1. Pan/Zoom the map to your needed location
  2. Use the context menu (right click/long tap on mobile) to add an hexagon
  3. Click on the hexagon to change it's size
  4. Add more hexagons and arrange them as you want
  5. Print the command line (nav at the top or context menu) and use it in PokemonGo-Map

Remove hexagon

  1. Right click (open context menu) on the hexagon
  2. Select "Remove hexagon"

Mobile

  • Use longtap instead of right click to open context menu
